# WhisperMe

![WhisperMe 대표 이미지](docs/images/whisperme-cover.png)

## 프로젝트 소개

WhisperMe는 OpenAI API를 활용한 AI 대화 서비스입니다.

텍스트와 이미지를 함께 분석할 수 있으며, 채팅방 관리와 할 일 관리, 날씨, 운세 등 다양한 기능을 제공합니다.

---

## 주요 기능

### AI 채팅
- GPT 기반 대화
- 이미지 분석
- 채팅방 관리
- 대화 기록

### 일정 관리
- 오늘 할 일
- 완료 체크
- 지난 일정 관리

### 생활 기능
- 날씨
- 운세
- 위치 기반 서비스

### 사용자
- 회원가입
- 로그인
- 프로필 수정

---
